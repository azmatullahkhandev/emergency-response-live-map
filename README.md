# Emergency Response Live Map

This project simulates an emergency response system using data structures to efficiently manage and route emergency requests.

## Features
- Simulates real-time emergency requests
- Optimized routing system
- Efficient handling of multiple requests

## Data Structures Used
- Graphs (for mapping locations)
- Queues / Priority Queues (for handling requests)
- Arrays / Linked Lists (if used)

## Applications
- Disaster management systems
- Ambulance dispatch systems
- Smart city solutions

## Technologies
- C++

## Note
This project was developed with the assistance of AI tools.

## How to Run

Compile:
g++ main.cpp -o map

Run:
./map
